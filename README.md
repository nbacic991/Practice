## MCI Practice

If you want to practice on this project, there are some steps you should need to follow.

### Step 1:

Open Your terminal window and type the following:

    docker-compose up -d
    

### Step 2:

Open Your browser and type in the address bar:

    practice.dev.loc/
    
And that is it.
